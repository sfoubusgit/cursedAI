STYLE-GUARD.md  
(MANDATORY — cursedAI)

All frontend UI, layout, copy, and interaction decisions MUST follow:
- design/DESIGN-cursedAI-v1.1.md
- planning/UX-cursedAI-v1.md

CORE LAYOUT RULES
- Feed MUST be single-column.
- ONE dominant media item per row.
- NO grids, NO galleries, NO cards, NO masonry.

RATING RULES
- Ratings ALWAYS apply to a specific media item.
- Rating prompt MUST visually reference the last viewed item.
- Never “rate the experience”.

TONE & VISUAL LANGUAGE
- Dark, muted, restrained.
- Subtle jester undertone (dry, observational).
- NO emojis, NO jokes, NO horror theatrics, NO glitch gimmicks.

DEFAULT RULE
If uncertain: choose the quieter, simpler option.

END.
